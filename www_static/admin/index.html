Login page

A list of addresses (protocol://domain[:optional port]) of the protected site. Requests from addresses that are not on the list are rejected.

If https://example.org is on the list and http://example.org is not, requests to http://example.org will be automatically redirected to https://example.org.

It is safe to add http://localhost[:port] to communicate with wwwhisper from a local machine.
